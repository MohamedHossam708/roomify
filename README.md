🧠 AI 2D → 3D Design Converter

An AI-powered web application that transforms 2D plans into fully visualized 3D designs — helping users better understand spaces, layouts, and design possibilities in a more intuitive way.

✨ Features
📤 Upload 2D plans and generate 3D designs using AI
🔄 Interactive before/after comparison slider
📥 Download final rendered designs بسهولة
🎨 Clean, minimal, and user-friendly interface
⚡ Fast performance powered by Vite
☁️ Serverless backend (Puter)
🏗️ Tech Stack
Frontend
Vite
React (or your framework — update if different)
Tailwind CSS (optional)
Backend (Serverless)
Puter
Authentication
File storage (images & designs)
Usage tracking & pricing
🚀 Getting Started
Prerequisites

Make sure you have installed:

Node.js (v18+ recommended)
npm or yarn
Installation
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate to the project folder
cd your-repo-name

# Install dependencies
npm install
Running the App
npm run dev

The app will be available at:

http://localhost:5173
Build for Production
npm run build
Preview Production Build
npm run preview
⚙️ Environment Variables

Create a .env file in the root directory and add:

VITE_PUTER_API_KEY=your_api_key_here

Replace with your actual Puter credentials.

📸 How It Works
Upload a 2D floor plan
AI processes the layout
Generates a 3D visualization
Compare before/after using slider
Download the final design
